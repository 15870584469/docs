          # 个人博客搭建

          # 准备工作

          下载nodejs和npm,nodejs自带npm nodejs:nodejs

          # vuepress概述

          • Vue 驱动的静态网站生成器
          • 基于markdown语法生成网页
          • 可自定义和扩展样式
          • 可以发布至github
          • 详情请看官网vuepress

          # 安装初始化

          • 全局安装
          $ npm install -g vuepress
          1
          • 创建个文件夹作为目录
          $ mkdir vuepress-blog
# 该目录作为整本书的项目目录
          1
          2
          • 项目初始化
          $ cd vuepress-blog
$ npm init -y
          1
          2
          • 在当前目录中创建一个docs目录
          $ mkdir docs
# 主要存放博客书籍内容
          1
          2
          • 首页内容书写(默认主题提供)
          ---
home: true
heroImage: /logo.jpg
actionText: 快速上手 →
actionLink: /zh/guide/
features:
- title: 简洁至上
  details: 以 Markdown 为中心的项目结构，以最少的配置帮助你专注于写作。
- title: Vue驱动
  details: 享受 Vue + webpack 的开发体验，在 Markdown 中使用 Vue 组件，同时可以使用 Vue 来开发自定义主题。
- title: 高性能
  details: VuePress 为每个页面预渲染生成静态的 HTML，同时在页面被加载的时候，将作为 SPA 运行。
