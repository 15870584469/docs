# 修炼谈话的艺术---《思考与成功》

# 摘 要:高超的谈话技巧是一门重要的艺术，但大多数人没有掌握这种艺术。善于谈话的人懂得，思想与思想之间的碰撞，心灵与心灵之间的沟通，会产生耀眼的火花和全新的力量。 一个高明

高超的谈话技巧是一门重要的艺术，但大多数人没有掌握这种艺术。善于谈话的人懂得，思想与思想之间的碰撞，心灵与心灵之间的沟通，会产生耀眼的火花和全新的力量。

一个高明的谈话者必定有他自己的想法，并能广泛的阅读，深入地思考和倾听他人的见解，因而总有某些可以谈论的内容。

当查尔斯。艾略特担任哈佛大学校长时，他说过这样一句话，“我认为有教养的青年男女唯一具有的必备素养，就是精确而优雅的使用本国语言。”

事实上，没有任何别的东西可以像良好的谈话能力那样能使别人，尤其是使那些对我们了解甚少的陌生人，对我们产生美好的印象。

成为一个好的健谈者，运用你在交流沟通方面非同一般的技能，就能够引起别人的兴趣，吸引他们的注意力，并自然使他们聚集在你的周围。这是一种非常重要的技能，其重要性无可比拟。它不但可以令你在陌生人心中留下美好的第一印象，同时也可以帮助你广交朋友，拥有友谊。它打开了人与人之间沟通的大门，使彼此的心灵变得亲近。它可以使你在各种各样的人群中广受欢迎，使得你能与他人融洽相处，在社会交往中如鱼得水。如果你拥有这样一种非同寻常的本领，那么客户、生意自然会源源而来。即便你在物质上一贫如洗，只要有良好的交谈能力做支撑，你照样可以进入上流社会。

一个人如果有良好的口才，掌握了一种以有生动有趣的方式来论述事物的艺术，并能借助滔滔不绝的口才来迅速的打动别人，那么，与那些知识比他更为丰富、却无法像他那样轻松自如的表达自己思想的人来说，他便拥有一种极大的优势。

不管你在其他方面的艺术或技能方面的专业造诣有多高，达到怎样一种炉火纯青的地步，但你肯定不可能像运用谈话技术一样随时随地地表现专业才能。如果你是一个音乐家，不管你的音乐天赋是如何了得，不管你花费了多少年的时间来提高自己的演奏技巧，也不管你耗费了多少金钱，却只有相对很少的一部分人可能听到或欣赏到你的音乐。

你或许是一个优秀的歌手，然而在你周游世界时，却没有机会向世人展示你的不凡技艺，或者没有人知道你的专业技能。但是，不管你到世界的哪个角落，不管你身处何种类型的社会，也不管你在生活中的定位如何，你都可以自由自在地交谈。

你可能是一位画家，可能孜孜不倦地跟着画坛的名家大师学习了很多年，然而，除非你拥有过人的天资，你的作品才可能被悬挂在沙龙里或者是在著名的艺术博物馆里供人欣赏。否则的话，你一辈子都可能默默无闻，并且，即便你是属于幸运的前者，也只有相对很少的一部分人才能看到你的作品。但是，如果你是一个谈话方面的艺术大师，任何一个跟你接触过的人都将看到你活生生的生命画卷。从你一开始谈话起，这幅画卷就活灵活现的被描绘了出来。通过谈话，任何人都会知道你到底是一位高明的艺术家，还是一个经验不足的拙劣人物。事实上，你可能拥有很多人只有偶尔看到或欣赏到的技艺，你可能有一个美丽温馨的家，有许许多多的财产，但是，所有的这些都只有相对极少的一部分人了解；然而，如果你是一个健谈者，那么任何一个与你交谈过的人都将强烈地领略到你的谈话艺术，并感受到你的魅力和影响力。
